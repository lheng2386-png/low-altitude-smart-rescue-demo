# Case 1 Flood Civilian Rescue

- Input source: local repository image candidates listed in `case_config.json`.
- Mask policy: manually prepared demo mask generated by `scripts/generate_demo_cases.py`.
- Segmentation note: the demo mask is for decision-layer demonstration. It is not an automatic segmentation prediction.
- Detection: uses real local YOLO detection when a local `models/yolov11*/best.pt` weight is available.
- Auto segmentation checkpoint: not required for this case.
- Current limitation: the generated route is an image-plane reference path, not a GPS route.
