Core Innovation

不是停留在检测框展示，而是把感知结果转成可解释的救援辅助决策。

TERP 联合优先级模型

融合目标类别、检测置信度、目标尺度、环境风险和路径可达性，输出综合救援优先级。

Risk-Aware A* 路径规划

对比普通 A* 与基于灾区 mask 代价地图的 Risk-Aware A*，降低路径经过高风险区域的比例。

Run Locally

交互式 AI Demo 仍然需要本地 Python / Gradio 环境。GitHub Pages 静态站只负责展示项目与案例。

git clone https://github.com/lheng2386-png/low-altitude-smart-rescue-demo.git
cd low-altitude-smart-rescue-demo/app
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

当前路径规划为图像平面参考路径，不等同于真实 GPS 路线；系统未接入真实道路网络、无人机定位或飞控系统。手工 demo mask 仅用于决策层展示，不是自动分割预测。
