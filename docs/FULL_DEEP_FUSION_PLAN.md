# Full Deep Fusion Plan

## Goal

Turn 灾情感知及影响评估 into a mature competition prototype by integrating platform workflow, detection assets, model comparison structure, and segmentation references into one decision-support loop.

## Required Fusion Layers

1. Platform layer: mission, upload, review, report, archive.
2. Detection layer: YOLOv11 image/video detection and gallery.
3. Segmentation layer: RescueNet-style classes, masks, reference gallery, optional checkpoint.
4. Decision layer: risk score, TERP, Scene Applicability Gate.
5. Access planning layer: ordinary A* vs risk-aware A*.
6. Reporting layer: Chinese rescue report.
7. Benchmark layer: model comparison module with reference/reproduced separation.
