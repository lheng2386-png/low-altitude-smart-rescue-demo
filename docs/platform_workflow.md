# Platform Workflow

```text
UAV Image / Video
→ Disaster Target Detection
→ Disaster-Scene Segmentation
→ TERP Priority Decision
→ Scene Applicability Gate
→ Risk-Aware Access Planning
→ Rescue Report
→ Platform-style Case Archive
```

Current implementation is a local prototype. Future work can add login, database storage, cloud deployment, GIS/GPS, and UAV flight-control integration.
