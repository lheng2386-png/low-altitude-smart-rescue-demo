- `POST_DISASTER_DATASET_README_REFERENCE.md`: post-disaster survivor dataset reference.

- `code/DINO_README_REFERENCE.md`: DINO detector README reference.

- `code/faster_rcnn_predict.py`, `code/faster_rcnn_train_res50_fpn.py`: copied Faster R-CNN inference/training structure.
